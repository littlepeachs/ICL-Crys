📊 实现的评估指标
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Table 1: 条件生成指标 (Conditional Generation)
────────────────────────────────────────────────────────────────────────────
对每个样本进行1000次采样，计算成功率的均值和标准差：

✓ Pretty Formula (Mean, Std)
  - 化学式匹配率
  - 检查生成结构的化学组成是否与目标一致

✓ Space Group (Mean, Std)
  - 空间群匹配率
  - 使用SpacegroupAnalyzer检查空间群编号

✓ Formation Energy (Mean, Std)
  - 形成能匹配率
  - 需要DFT计算（提供占位符接口）

✓ Band Gap (Mean, Std)
  - 带隙匹配率
  - 需要DFT计算（提供占位符接口）

Table 2: 无条件生成指标 (Unconditional Generation)
────────────────────────────────────────────────────────────────────────────
生成10,000个样本，计算以下指标：

1. Validity Check (有效性检查)
   ✓ Structural Validity - 原子间距离 > 0.5Å
   ✓ Compositional Validity - 电荷中性检查
   ✓ Total Validity - 综合有效性

2. Coverage (覆盖率)
   ✓ Recall - 覆盖了多少参考结构
   ✓ Precision - 生成的有效结构比例

3. Property Distribution (属性分布)
   ✓ Density - Wasserstein距离
   ✓ Number of Atoms - Wasserstein距离
   ✓ Volume - Wasserstein距离
   ✓ Number of Elements - Wasserstein距离

2. 测试模块:
   python test_modules.py

3. 训练模型:
   python run_crystalicl.py \
       --model_name Qwen/Qwen3-8B \
       --use_sample_data \
       --do_train \
       --num_epochs 3

       --num_samples 1000 \
       --num_unconditional 10000

5. 查看结果:
   cat evaluation_results_complete.json

🔧 关键技术特性
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

模型架构:
  • 基础模型: Qwen3-8B (128K context)
  • 微调方法: LoRA (Rank=8, Alpha=32)
  • 训练精度: BF16混合精度
  • 参数效率: 只训练0.1%参数

Token化方法:
  • SGS格式: 基于空间群和Wyckoff位置
  • 减少token数量: 相比XYZ格式减少50%+
  • 保留对称性: 完整的晶体对称信息

示例选择策略:
  • Condition-based (C): 基于属性过滤
  • Structure-based (F): 基于结构相似度
  • Hybrid (CF): 混合策略（推荐）

评估体系:
  • 条件生成: 4个指标（Mean + Std）
  • 无条件生成: 3类指标（8个子指标）
  • 完整覆盖: 论文Table 1 & 2所有指标
