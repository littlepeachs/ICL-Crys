🎯 核心功能实现
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Space-group based Crystal Tokenization (SGS) ✅
   • 将3D晶体结构转换为1D文本序列
   • 基于Wyckoff位置简化对称性建模
   • 相比XYZ格式减少token数量50%+

2. Condition-Structure Aware Hybrid Instruction Tuning ✅
   • 零样本指令生成
   • 少样本指令生成（K-shot）
   • 三种示例选择策略：
     - Condition-based (C)
     - Structure-based (F)
     - Hybrid (CF) ⭐ 最佳性能

3. Multi-Task Crystal Instruction Tuning ✅
   • 晶体生成任务
   • 属性预测任务
   • 联合训练提升泛化能力

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

模型架构:
  • 基础模型: Qwen2.5-7B-Instruct (替代Llama-2)
  • 微调方法: LoRA (Rank=8, Alpha=32)
  • 训练精度: BF16混合精度
  • 上下文长度: 32K tokens

训练优化:
  • 梯度累积: 8步
  • 学习率: 5e-4 (cosine调度)
  • 优化器: AdamW
  • 只训练0.1%参数（LoRA）

数据支持:
  • JSON格式
  • CIF文件批量加载
  • 自动数据集划分
  • 示例数据生成器

💡 使用示例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Python代码:
  from train_crystalicl import CrystalICLTrainer

  trainer = CrystalICLTrainer(model_name="./crystalicl_qwen_output")

  instruction = """### Instruction: The chemical formula is NaCl.
  Generate the crystal structure:
  ### Response:"""

  generated = trainer.generate(instruction)
  print(generated)

命令行:
  python run_crystalicl.py \
      --data_path ./your_data.json \
      --do_train \
      --do_eval \
      --num_epochs 3 \
      --k_shot 3
